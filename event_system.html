Event System

Overview

Morphyn uses an event queue to process entity reactions. Events are processed in order, and each event can trigger additional events.

Built-in Events

Init Event

Called when an entity is first created or spawned:

entity Enemy {
has hp: 50
on init {
emit log("Enemy spawned")
}
}

tick(dt)

Called every frame with delta time in milliseconds:

entity Timer {
has time: 0
on tick(dt) {
time + dt -> time
}
}

destroy

Marks entity for garbage collection:

entity Enemy {
has hp: 50
on damage(v) {
hp - v -> hp
check hp <= 0: emit self.destroy
}
}

Custom Events

Define your own events:

entity Player {
on jump {
emit log("Player jumped!")
}
on heal(amount) {
hp + amount -> hp
}
}
